Effects Implementation-Christina Tagay

Effect is implemented by creating an Effect base class that is derived from CInstrument (which is derived from CAUdioNode) so I'm able to access the protected attributes of CAudioNode. All effects in their own class CEffect will have access to these attributes in Process method. This process method will be called if it was read in from the score files. CSynthesizer will have boolean variables set to true if found in the file and will call the effect's Process function if true.

Grading Criteria

  • Component passes audio
  • 1 Effect
  • 3 Effects

Score Format

        <!-- EFFECTS -->
        <instrument instrument="Reverb">
                <note measure="3" delay=".0025" wet="1" dry="1" reverbTime="0.5" />
                <note measure="4" delay=".0025" wet="1" dry="1" reverbTime="0.7" />
                <note measure="5" delay=".0025" wet="1" dry="1" reverbTime=".5" />
        </instrument>
        <instrument instrument="Chorus">
            <note measure="3" beat="3" delay=".5" wet=".06" dry=".5" />
            <note measure="4" beat="3" delay=".5" wet=".08" dry="0" />
            <note measure="5" beat="3" delay=".5" wet=".05" dry=".3" />
        </instrument>
        <instrument instrument="Flange">
            <note measure="6" delay="0.25" wet="0.8" dry="0.2" />
            <note measure="7" delay="0.25" wet="0.7" dry="0.3" />
            <note measure="8" delay="0.25" wet="0.6" dry="0.4" />
        </instrument>

This is an example of Reverb. If I want to add an effect, I create a new instrument with the instrument attribute as the effect (in this example Reverb). Similar to Subtractive, I create a note tag and write which measure and beat I want the effect to play. Unlike Subtractive or (other instruments) I do not include a note or duration of the effect Delay represents delay variance, wet represents wet, dry represents dry (both used to calculate output frame). But reverb has attribute reverbTime. This represents how long the 'echo' sounds last as the sound decays. Reverb is the only effect with this attribute. Chorus and Flange have the same attributes as Reverb besides ReverbTime.
